# Explorer — Kachelansicht mit Bildern

**Status:** Draft
**Datum:** 2026-04-20

## Anforderung

Im Trend-Explorer eine zusätzliche Kachelansicht (Tile/Grid View) mit Bildanzeige pro Trend.

## Verhalten

1. Toggle zwischen bestehender Listen-/Grid-Ansicht und neuer Kachelansicht
2. Kachel zeigt: Trend-Bild (gross), Titel, Dimension, Stage
3. Abhängig von Trend-Bild Feature (siehe `2026-04-20-trend-image.md`)

## Kontext

- Betrifft: `src/explorer.jsx` (hat bereits Grid/List Toggle)
- Dritte Ansichtsoption: List / Grid / Tiles
- Bilder kommen aus dem Trend-Bild Feature (manuell oder KI-generiert)
